Installing Oreo

Note

This tutorial is written for Windows machines!

Step 1

Before anything, make sure you have a suitable IDE installed (I recommend VSCode!), along with the NodeJS runtime.

To check if you have NodeJS installed, enter a terminal and type node. In the IDE of your choice, open a folder for Oreo (On VSCode, this is ctrl + K + ctrl + O). In that folder, open your terminal (For VSCode, ctrl + ').

If you have git installed, simply type: git clone https://github.com/saaawdust/oreolang into your terminal, followed by cd oreolang. Now continue from step 3.

If you do not have git installed, head over here, and click the code button, followed by Download Zip (Sorry for the horrendous image quality, no idea what happened there):

Step 2

Unzip the contents of Oreo into your folder (cwd) and it should look something like this (Don’t worry if node_modules isn’t there, we’ll get to that in a second):

Step 3

Simply install all dependencies with npm install, and you’re almost done. After running npm run setup, Oreo should be installed on your machine (as long as you don’t delete the folder!)
